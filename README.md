# Danni Xu Academic Homepage

A minimal academic homepage for GitHub Pages.

## How to deploy

1. Create a GitHub repository named:

   ```text
   dannixu.github.io
   ```

2. Upload these files to the repository root:

   - `index.html`
   - `style.css`
   - `script.js`
   - `assets/` folder

3. In GitHub, go to:

   Settings → Pages → Build and deployment → Deploy from a branch

4. Select:

   - Branch: `main`
   - Folder: `/root`

5. Your homepage will be available at:

   ```text
   https://dannixu.github.io
   ```
